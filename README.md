# credit_card_fraud_detection
machine learning project for detecting fradulent credit card transactions using logistic regression and decision tree  classifier
    credit card fraud causes significant financial losses for banks and financial institutions and i build a machine learning model that can detect frudulent card transactions 
    the dataset cointains credit card transactions with a binary target variable 0: legitimate transaction and 1:fraudlent transaction
    evaluation metrics such as confusion matrix,recall,f1 score,precision,roc curve,auc score
    logistic regression performs better than decision tree
    logistic regressions provides stronger fraud detection capabilityv and is recommended for deployment
